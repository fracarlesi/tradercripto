1. COOLDOWN SYSTEM (Reactive Safety)
   ─────────────────────────────────────────────────────────────────────
   Auto-pauses trading after adverse conditions:

   • 3 Triggers:
     - 3+ consecutive stoplosses in 1h → 6h cooldown
     - Daily drawdown > 5% → 12h cooldown
     - Win rate < 20% on 5+ trades → 24h cooldown

   • Features:
     - Database persistence (survives restarts)
     - Telegram alerts
     - Dashboard indicator
     - API: /api/cooldown-status

   • Test Coverage: 20 tests ✅


2. PERFORMANCE METRICS (Risk-Adjusted Tracking)
   ─────────────────────────────────────────────────────────────────────
   Tracks 8 professional trading metrics:

   • Risk-Adjusted Returns:
     - Sharpe Ratio (annualized)
     - Sortino Ratio (downside deviation)
     - Calmar Ratio (return/max drawdown)

   • Drawdown Tracking:
     - Max Drawdown % (peak-to-trough)
     - Max Drawdown $ (absolute)
     - Current Drawdown %

   • Trade Quality:
     - Profit Factor (gross profit/gross loss)
     - Win Rate
     - Expectancy (Van Tharp)

   • System Quality:
     - SQN (System Quality Number)

   • Features:
     - Real-time calculation
     - Dashboard card with color coding
     - API: /api/performance-metrics

   • Test Coverage: 41 tests ✅


3. GRADUATED ROI (Smart Profit-Taking)
   ─────────────────────────────────────────────────────────────────────
   Time-based take profit thresholds:

   • 6 Thresholds:
     - 0-30min:   3% profit target
     - 30-60min:  2% profit target
     - 1-2h:      1.5% profit target
     - 2-4h:      1% profit target
     - 4-8h:      0.5% profit target
     - 8h+:       Break-even (any profit)

   • Benefits:
     - Captures early profits when available
     - Lets trades breathe for larger moves
     - Break-even protection after 8h
     - Prevents loss reversal

   • Implementation:
     - Works for long AND short positions
     - Integrated in position monitoring loop
     - Detailed exit logging

   • Test Coverage: 20 tests ✅


4. PROTECTION SYSTEM (Proactive Safety)
   ─────────────────────────────────────────────────────────────────────
   4 modular protections that block trading:

   • StoplossGuard:
     - Trigger: 3+ SL in 1h → Block 6h

   • MaxDrawdownProtection:
     - Trigger: Drawdown > 5% in 24h → Block 12h

   • CooldownPeriodProtection:
     - Trigger: Trade within 5 min → Block until elapsed

   • LowPerformanceProtection:
     - Trigger: WR < 30% on 20 trades → Block 24h

   • Architecture:
     - Abstract Protection base class
     - ProtectionManager orchestrator
     - Database persistence
     - Telegram alerts
     - Dashboard UI
     - API: /api/protections

   • Test Coverage: 28 tests ✅
